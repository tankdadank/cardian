## 3. Data Collected

### User Account Data

- Discord user ID  
- Display name (for gameplay rendering)  
- Patreon tier snapshot  
- Cosmetic preferences  
- Profile settings  
- Saved presets (if used)  
- Match history summary  
- Last active session reference  

### Gameplay Data

During active matches:

- Session ID  
- Game type  
- Player IDs  
- Deck state  
- Turn state  
- Rule snapshot  
- Match results  
- Limited action logs  

AI mirror data is temporary and not retained long-term.

### Server Configuration Data

- Guild ID  
- Owner ID  
- Enabled game flags  
- Allowed channels  
- Timeout settings  
- SVS configuration  
- Role ID references  
- Experimental flags  

### Moderation & Integrity Logs

- Moderator ID  
- Target user ID  
- Action type  
- Timestamp  
- Reason (if provided)

Moderation records are not publicly visible and are not individually deletable due to integrity requirements.

### Web & OAuth Data

If using the companion website:

- OAuth scopes  
- Temporary session tokens  
- Dashboard access flags  
- Webhook configuration (if enabled)

Session tokens expire automatically.

## 4. How Data Is Used

Data is used only to:

- Operate gameplay systems  
- Enforce deterministic rules  
- Maintain rankings  
- Restore sessions after crashes  
- Prevent abuse and fraud  
- Enable supporter features  
- Operate the companion dashboard  

Data is not used for advertising or automated decision-making with legal or significant effects.

---

## 5. Data Retention

### Operational Retention

Standard operational data may be retained up to 5 days where short-term limits apply.

Enthusiast tier (if enabled) may allow up to 10 days where extended retention applies.

Persistent account elements such as rankings and achievements remain until user deletion or service discontinuation.

### Backups

- Generated at the beginning of each month  
- Encrypted  
- Retained up to 90 days  
- Automatically expire  

Deletion from active systems does not immediately remove data from existing encrypted backups.

Minimal records may be retained where necessary for fraud prevention, abuse detection, or legal claims defense.

### Logs

Access logs may include:

- IP address  
- Browser metadata  

Retained up to 7 days.

Internal system logs are not publicly visible.  
Moderation-facing logs may be partially anonymized.
